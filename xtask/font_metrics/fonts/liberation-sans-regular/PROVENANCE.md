## Generation Intent

`mmdflux-sans-v1` is an mmdflux-owned deterministic metrics profile. Liberation
Sans Regular is source provenance only; the persisted profile ID must not imply
browser, Mermaid, or system-font identity.

The raw TTF is not vendored in this task. Task 1.2 owns adding or otherwise
resolving source font bytes for generation and then verifying package impact.
